Policies

Originally a dog lover community is on social network to people who have the same interest for exchange information and include center of related activities. Because in present there are currently many social media users. Its makes crime to provide incorrect information to users in this social network (include write fake news, sale fake product and created fake event) for money and this situation is very dangerous because people who tricked by crime can’t check the person responsible. And can’t confirm security to participle in various activities.

With this reason a community leader make sure you have a good website that you can use to solve the problem. A group of community leaders have an infrastructure system. Such as Web Server, Database Server and Domain name registered and the investment for this project is 3 million baht and want the system is ready for use on May 1, 2017.

Requirement

  • User should divided into 4 type (include Site visitor, Member, VIP Member, Administrator).
  • Member and Starred members should registered with own ID card number but can set username and password for use this website.
  • Only one administrator can grant membership.
  • After membership Users who create benefits to the community that meet the criteria will be elevated. Admin will promote status form member to a starred member.
  • The system must keep the personal information of each member securely. Not accessible if not allow form admin because this information is used for monitoring and tracking only and not disclosed to the public.
  • Community homepage must show the main menu, news, about, event information.
  • Visitors to the website can view the main menu, news, about, event information only. But can’t join the event or write a comment on content that is readable.
  • All members can write and publish their own articles publicly, or only members can comment on other members' articles and can purchase tickets to participate in community activities.
  • Starred User and Administrators only have the right to organize events for the community.
  • Community events activities are free and tickets are required to attend the event.
  • Starred User can invite people to become members, if the members are starred members. Payments required activities Get 10% off the ticket price to join the event up to 3 times a year.
  • If starred members make community abuse or create trouble for the community. Administrators are entitled to reduce the member's level. Members can also be revoked.
  • The system must accommodate up to 5,000 simultaneous connections, responding back in time. Up to 5 seconds.
  • The system must be highly secure. To ensure the user.
  • The system must work 24 hours with Downtime up to 15 minutes per session, up to 1 time per week.
